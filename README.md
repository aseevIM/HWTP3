# Инструкция для "лютого" задания

1) Запускаем команду ./run.sh report_server ПОСЛЕ ТОГО как мы сгенерили csv и отчет!
2) В панельке рядом с терминалом справа будет кнопка "Порты"
3) Там будет строка с портом 8080, и после наведения на строку будет значок браузера, нажимаем на него
4) Потом выйдет страница с ошибкой 403(мой опыт) и в конце ссылки браузера добавляем /report.html
5) Запускаем и победа!

Сразу дам небольшое обьяснение. Эта команда report_server создает третий контейнер(как и в подсказке) на основе образа nginx:alpine. Затем при запуске контейнера я пробросил порт через флаг -p 8080:80, и поэтому порт 80 виден как 8080 на портах кодспейса гитхаба. А потом мы используем флаг -v "$(pwd)/data:/usr/share/nginx/html:ro" и папка data с нашего хоста привязывается в папку, откуда nginx по умолчанию берёт файлы для показа. На всякий написал :ro, чтобы контейнер только читал файлы. Вроде все (◕‿◕)<3
